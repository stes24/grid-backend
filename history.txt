- Creato ambiente virtuale: python -m venv venv (contiene una copia di Python
  e i pacchetti installati solo per questo progetto, invece che globalmente)
- Attivato ambiente virtuale: .\venv\Scripts\Activate.ps1 (pip verrà usato lì),
  si disattiva con deactivate
- Installati pacchetti: pip install flask psycopg2-binary python-dotenv
  - flask: web framework minimalista usato per esporre API REST
  - psycopg2-binary: driver PostgreSQL (fa da tramite per l'interazione col db)
  - python-dotenv: per caricare variabili d'ambiente da .env (per connettersi al db,
    segrete, da non caricare su git)
- Salvato l'elenco dei pacchetti installati: pip freeze > requirements.txt
  (serve per ricreare l'ambiente virtuale su un'altra macchina, simile a package.json:
  python -m venv venv, .\venv\Scripts\Activate.ps1, pip install -r requirements.txt)
  -> da rieseguire ogni volta che installi un nuovo pacchetto
- Impostate variabili d'ambiente in .env per connettersi al db

- Installato flask-cors per permettere comunicazione tra porte diverse

- Sviluppata app come endpoint REST per leggere/aggiornare i pixel (presenta delle route
  a cui si collegherà il client), avviata nell'ambiente virtuale: python app.py
